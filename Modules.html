Ansible Modules

  1. Copy files to default apache2 directory.

    copy
      src: index.html
      dest: /var/www/html/index.html
  2. Start apache2 service.

    service
      name: apache2
      state: started
      enabled: yes
  3. Create user.

    user
      name: mreddy
      state: present
  4. Create wheel group.

    group
      name: wheel
      state: present
  5. Add user to the wheel group.

    user
      name: mreddy
      group: wheel
      append: yes
  6. Based on OS family installing git.

    yum
      name: git
      state: present
    when: ansible_os_family == "RedHat"
  7. Based on OS family installing git.

    apt
      name: git
      state: present
    when: ansible_os_family == "Debian"
  8. Debug module.

    debug
      msg: " The os family is {{ ansible_os_family }} "
  9. Debug module.

    debug
      var: version
  10. Command module.

    command: echo "My name is mallikarjun"
